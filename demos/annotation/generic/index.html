Redlink Annotation Demo

Give this standard Redlink engine a try! Add content and watch the annotations appear. This a standard setup of the Redlink enhancement engine, which can be improved by customizing the vocabularies used. So please, feel free to signup to access a more advanced experience.

This demo uses the Redlink Analysis API to annotate content with open knowledge datasets (DBpedia and Freebase). If you have any question, please, do not hesitate to contact us.
